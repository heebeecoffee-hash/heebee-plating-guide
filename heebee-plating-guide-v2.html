◈ How to Deploy

1. Open script.google.com → New Project
2. Paste the contents of heebee-plating-backend.gs
3. ⚙ Project Settings → Script Properties → Add:
    OPENAI_KEY · ANTHROPIC_KEY · IMAGE_MODEL · IMAGE_QUALITY
4. Deploy → New deployment → Web App
    Execute as: Me · Who has access: Anyone
5. Copy the /exec URL → paste above
